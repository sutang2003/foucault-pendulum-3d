# Foucault Pendulum 3D Visualization

This project demonstrates the precession of a Foucault pendulum using an interactive 3D model.

## Demo
https://www.geogebra.org/3d/auwcx8x6
https://www.youtube.com/watch?v=EXEZUVuL5mo

## Description
- Built an interactive model to visualize Earth's rotation effect
- Translated abstract physics concepts into intuitive visualization
- Designed for user interaction and exploration
